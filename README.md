# Cell Phone Store

An app to record all store's transactions data and reports. ERD (Entity Relationship Diagram) is provided [here](https://drawsql.app/teams/rand/diagrams/cell-phone-store).

## Tech Stack & Libraries

-   Tech Stack
    -   [Laravel](https://laravel.com/docs/10.x) as API Service
    -   [React JS](https://reactjs.org/) as Client Service

<br/>

-   Libraries
    -   Laravel
        -   [Json Web Token](https://github.com/tymondesigns/jwt-auth) by Sean Tymon
        -   [Laravel Permission](https://spatie.be/docs/laravel-permission/v5/introduction) by Spatie
    -   React

<br/>

## Features

-   Managing
    -   Users
    -   Categories
    -   Brands
    -   Permissions
-   Record Transactions
-   Reporting
